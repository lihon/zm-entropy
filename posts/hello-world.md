这是博客的第一篇文章。就像每个程序员的第一行代码一样，从这里开始。

## 为什么要写博客

写作是最好的思考方式。当你把想法写下来，那些模糊的概念会变得清晰，零散的知识会形成体系。

写博客不是为了给别人看，而是为了：

- 整理自己的思路
- 记录学习的过程
- 积累可以回顾的知识
- 在写作中发现新的理解

## 博客搭建

这个博客没有使用任何框架或静态网站生成器。它就是纯 HTML + CSS，不到 200 行代码。

为什么选择这么简单的方式？

> 简单是最高级的复杂。—— 达·芬奇

不需要数据库，不需要 JavaScript 框架，不需要构建工具。写文章就是创建一个 Markdown 文件，编辑内容，推到 GitHub 上。一切都很直接。
